## Explainer

This is just a website displaying F1 stats, primarily focused on the current F1 championship year. This website is entirely done with Svelte/Sveltekit as it is my favorite frontend framework of choice with the [jolpica-f1](https://github.com/jolpica/jolpica-f1) for the API. Due to jolpica's current rate limitations and a desire for the website to be seamless and quick, I've decided to not load historical driver data and have race data only down to 2020. I respect jolpica's decision as they're currently still working on a general release for their API and as such, the website has limited functionality to this year to not exceed their rate limitations.
